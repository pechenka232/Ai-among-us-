Коротко
------
Локальная симуляция Among Us с ботами (мирные / предатель), визуализацией карты и вызовами ИИ через OpenRouter. Предатель (Фиолетовый) логирует все свои промпты/ответы в purple_thoughts.log.

Требования
----------
- Python 3.9+
- Библиотеки:
  - pygame
  - requests

Установка
---------
```bash
pip install pygame requests
```

Где взять ключ OpenRouter
-------------------------
1. Перейдите на https://openrouter.ai — зарегистрируйтесь (или войдите).
2. В личном кабинете найдите раздел API / Keys и создайте новый API-ключ.Можно использовать любую ИИ я в основу взял какой то там чат гпт 20obbs вроде
3. Скопируйте ключ и задайте его перед запуском приложения как переменную окружения:


Запуск

```bash
python main.py
```

Коротко о логах и победе
------------------------
- Лог мыслей предателя: purple_thoughts.log (в папке запуска).
- Условия победы реализованы: если предатель выброшен — победа мирных; если предатель остался и все мирные мертвы — победа предателя.
- Задания в игре — для прикола и вайба:3 и не влияют на победу основная задача скинуть преда но ИИшки тоже по приколу бегают задания делают.
